Index

Phantom supports 4 LLM providers out of the box. You can use one or configure all of them ‚Äî Phantom will auto-detect the best available provider.

Priority Order

When you run phantom without specifying a provider, it checks in this order:

  1. Ollama (if running locally)
  2. OpenAI (if OPENAI_API_KEY is set)
  3. Anthropic (if ANTHROPIC_API_KEY is set)
  4. Gemini (if GEMINI_API_KEY is set)

Configuration

Environment Variables

export OPENAI_API_KEY="sk-..."
export ANTHROPIC_API_KEY="sk-ant-..."
export GEMINI_API_KEY="AIza..."

Config File

phantom config setup

This stores keys in ~/.phantom/config.json.

Switching Providers

phantom chat --provider openai --model gpt-4o

phantom (ollama:llama3.1) ‚ñ∏ /model gpt-4o

Fallback Chain

If your primary provider fails (rate limit, network error), Phantom automatically falls back through the chain: OpenAI ‚Üí Anthropic ‚Üí Ollama ‚Üí Gemini.
