Gemini

Connect Gemini models via Google AI Studio for fast, cost-effective AI.

Setup

  1. Get an API key from aistudio.google.com.
  2. Set the environment variable:
    export GEMINI_API_KEY="AIza..."

Models

ModelContextBest ForCost
gemini-2.0-flash1MSpeed, general tasksFree tier available
gemini-2.5-pro1MComplex reasoning~$1.25/M tokens
gemini-1.5-pro2MVery long context~$1.25/M tokens
gemini-1.5-flash1MFast, cheap~$0.075/M tokens

Usage

phantom chat --model gemini

Notes

  • Gemini 2.0 Flash has a generous free tier.
  • The 1M+ token context window makes it excellent for analyzing large codebases or documents.
