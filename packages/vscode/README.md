# Phantom VS Code Extension

The official VS Code extension for Phantom - The AI-native PM Operating System.

## Features

- **Dashboard**: Access your local Phantom dashboard (localhost:3333) directly within VS Code.
- **Commands**: Generate PRDs, view stats, and manage your project.
- **Status Bar**: Live project context and agent status.

## Requirements

- Node.js 18+
- Phantom CLI installed (`npm install -g @phantom-pm/cli` or bundled)

## Usage

1. Open the Phantom sidebar icon.
2. View your project dashboard.
3. Use `Cmd+Shift+P` -> `Phantom: Generate PRD`.
