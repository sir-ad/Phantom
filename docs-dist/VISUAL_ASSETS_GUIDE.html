VISUAL_ASSETS_GUIDE

PHANTOM Visual Assets Guide

This document provides specifications and guidance for creating visual assets for the PHANTOM brand.

Brand Identity

Colors

Primary Palette:

  • Matrix Green: #00FF41 - Primary brand color, used for highlights and accents
  • Matrix Dark: #0D1117 - Background color, dark theme base
  • Phantom Orange: #FF6B35 - Secondary accent for CTAs and important elements
  • Cyber Blue: #00D4FF - Tertiary accent for variety and depth

Neutral Palette:

  • White: #FFFFFF - Primary text on dark backgrounds
  • Gray 100: #F5F5F5 - Light backgrounds
  • Gray 400: #9CA3AF - Secondary text
  • Gray 600: #4B5563 - Tertiary text
  • Gray 800: #1F2937 - Dark elements
  • Black: #000000 - Pure black for contrast

Typography

Primary Font: Inter (Sans-serif)

  • Clean, modern, highly readable
  • Used for body text and UI elements

Monospace Font: JetBrains Mono or Fira Code

  • Used for code blocks and terminal elements
  • Maintains the technical/developer aesthetic

Logo Specifications

Logo Variants

  1. Full Logo (with wordmark)

    • Format: SVG, PNG
    • Sizes: 1000x1000px, 500x500px, 200x200px
    • Usage: Website header, marketing materials

  2. Icon Only (ASCII art symbol)

    ░█▀█░█░█░█▀█░█▀█░▀█▀░█▀█░█▄█
░█▀▀░█▀█░█▀█░█░█░░█░░█░█░█░█
░▀░░░▀░▀░▀░▀░▀░▀░░▀░░▀▀▀░▀░▀
    • Usage: Favicon, app icon, small spaces

  3. Horizontal Logo

    • Icon + wordmark side by side
    • Usage: GitHub README, documentation headers

Logo Colors

  • Primary: Matrix Green on transparent/dark background
  • Inverse: White or Black on colored backgrounds
  • Monochrome: Single color for restricted use cases

Required Assets

1. Logo Files

Files to Create:

  • phantom-logo.svg - Vector master file
  • phantom-logo.png (1000x1000px)
  • phantom-logo-icon.svg
  • phantom-logo-icon.png (512x512px)
  • phantom-logo-horizontal.svg
  • phantom-logo-white.svg (for dark backgrounds)

Location: docs/assets/

2. Banner Image

Specifications:

  • Size: 1200x630px (Open Graph standard)
  • Format: PNG
  • Content: Logo + tagline + subtle Matrix rain effect
  • Usage: Social sharing, GitHub social preview

Design Elements:

  • Dark background (#0D1117)
  • Matrix Green accent elements
  • Logo centered or left-aligned
  • Tagline: "The invisible force behind every great product"
  • Subtle animated Matrix rain (for web)

3. Demo GIF

Specifications:

  • Size: 800x600px (or 1200x800px for higher quality)
  • Format: GIF or MP4 (looping)
  • Duration: 15-30 seconds
  • File size: < 5MB for GIF, < 10MB for MP4

Content Flow:

  1. 0-3s: Boot sequence showing PHANTOM loading
  2. 3-8s: Type command phantom prd create "User Auth"
  3. 8-15s: Show PRD generation progress and completion
  4. 15-20s: Type command phantom swarm "Should we add OAuth?"
  5. 20-30s: Show agent analysis and consensus result

Tools:

  • Recording: asciinema, terminalizer, or screen recording
  • Editing: After Effects, ScreenFlow, or simple GIF tools
  • Optimization: gifsicle, ffmpeg

  • Transparent background

Design Principles

1. Matrix Aesthetic

  • Green glow effects on important elements
  • Monospace fonts for technical authenticity
  • Terminal-style code blocks and UI elements
  • Dark backgrounds with subtle gradients
  • Animated elements (Matrix rain, cursor blinks)

2. Cyberpunk Elements

  • Neon accents (Matrix Green, Cyber Blue)
  • Grid patterns or circuit board motifs
  • Holographic effects (subtle)
  • Futuristic typography

3. Clean & Professional

  • Ample white space
  • Clear hierarchy
  • Consistent spacing
  • Readable text sizes

Asset Creation Workflow

Tools Recommended

Vector Graphics:

  • Adobe Illustrator
  • Figma (recommended for collaboration)
  • Inkscape (free alternative)

Raster Graphics:

  • Adobe Photoshop
  • GIMP (free alternative)
  • Affinity Photo

Animation:

  • Adobe After Effects
  • ScreenFlow (Mac)
  • OBS Studio + editing tools

Terminal Recording:

  • asciinema
  • terminalizer
  • screen recording software

File Naming Convention

phantom-{type}-{variant}-{size}.{ext}

Examples:
- phantom-logo-full-1000x1000.svg
- phantom-banner-og-1200x630.png
- phantom-demo-prd-generation.gif
- phantom-screenshot-dashboard-1440x900.png

Export Settings

SVG:

  • Clean paths
  • Outlined fonts (or include font links)
  • Optimized (use SVGOMG)

PNG:

  • 300 DPI for print
  • 72 DPI for web
  • Transparent background where appropriate
  • Compressed (use TinyPNG)

GIF:

  • 256 colors max
  • Optimized palette
  • Loop forever
  • < 5MB file size

MP4:

  • H.264 codec
  • 1080p or 720p
  • Optimized for web
  • Looping playback

Quality Checklist

Before using any asset:

  • Correct dimensions
  • Proper file format
  • Optimized file size
  • Consistent branding
  • Readable text
  • No placeholder content
  • Tested on target platforms
  • Backup files saved
