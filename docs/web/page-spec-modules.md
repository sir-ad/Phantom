# Page Spec: Modules

Owner: PhantomPM Web + Platform Team  
Last Updated: 2026-02-13  
Status: Planned

## Goal

Show module ecosystem value while maintaining trust and compatibility clarity.

## Sections

1. module catalog grid
2. filter by category and status
3. install command snippets per module
4. compatibility and dependency notes
5. quality/trust metadata

## Module Card Requirements

1. name and short description
2. quote/theme line
3. version
4. status (`Real/Beta/Planned`)
5. install command
6. docs link

## Acceptance Criteria

1. module install snippets match CLI contract
2. no module marked `Real` without implementation path
3. dependencies and prerequisites displayed
