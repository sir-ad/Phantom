# Web Design System

Owner: PhantomPM Design Team  
Last Updated: 2026-02-13  
Status: Planned

## Brand Direction

Matrix-cyberpunk identity with practical readability and credibility.

## Color Tokens

1. `--bg-primary: #0D1117`
2. `--bg-secondary: #161B22`
3. `--text-primary: #E6EDF3`
4. `--text-secondary: #8B949E`
5. `--matrix-green: #00FF41`
6. `--cyber-blue: #00D4FF`
7. `--phantom-orange: #FF6B35`
8. `--neon-red: #FF2D55`

## Typography

1. display/headline: monospace expressive family
2. body: high-legibility sans-serif
3. code/install snippets: monospace with strong contrast

## Motion Principles

1. purposeful motion only (entry transitions, demo sequencing)
2. avoid distracting continuous effects on critical CTA blocks
3. support reduced-motion preferences

## Components

1. Install command block with copy interaction
2. Status badge component (`Real/Beta/Planned`)
3. Terminal transcript container
4. Feature card
5. Integration card
6. Docs callout block

## Accessibility Requirements

1. WCAG AA contrast minimum
2. keyboard operable controls
3. visible focus states
4. semantic structure for screen readers

## Content Style

1. high-energy but clear
2. avoid unsupported absolutes
3. always pair bold claims with evidence links

## Acceptance Criteria

1. design tokens are centralized and documented
2. status badge usage is consistent across pages
3. CTA components are accessible and testable
